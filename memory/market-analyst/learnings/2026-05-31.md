## 4. 틀린 것 / 개선 필요 사항

### (b) Non-Spin overnight (HE01-06) AS 기술 누락 — 7사이클
오늘(2026-06-01) 브리핑도 AS 섹션을 HE20-22 중심으로만 기술. Non-Spin HE01-06 ($2.02/MWh at HE01, $3.13/MWh at HE02)의 overnight AS 스택이 다시 누락됨. 절대값 낮으나 BESS 스케줄 최적화에 참고 가능한 정보이므로 AS bullet에 한 줄 추가 반드시 필요 — 이번에도 미완료.

### (c) ECRS morning ramp (HE07-10) AS 기술 누락 — 7사이클
동일. 2026-06-01 데이터: HE07 ECRS $0.49, HE08 ECRS $0.86 — 낮은 절대값이지만 구조적 표준화가 필요. P2 항목이었으나 계속 밀리고 있음. 다음 사이클에서 P1으로 격상.

### (d) Enverus net-load 갭 드라이버 분해 실행 — 완료 (개선)
오늘 브리핑에서 Enverus net-load avg(42,512 MW) vs YE(40,156 MW) +2,355 MW 갭을 분해 시도: Enverus solar avg(12,250 MW) = YE(12,700 MW) −450 MW, Enverus wind avg(10,971 MW) = YE(11,103 MW) −131 MW. Load avg 갭 +203 MW. 수지: load 갭 +203 MW − solar 갭 −450 MW − wind 갭 −131 MW = +784 MW 불일치. 잔여 ~1,571 MW 갭은 Enverus net-load FC 산출 방식(저장장치, 비전통 자원 등)의 차이로 추정. 분해는 시도했으나 완전 설명 불가 — 향후 Enverus 방법론 확인 필요.

### (e) AG2 데이터 파싱 완료 미달성 — 재발
어제 브리핑에서 처음으로 AG2 windcast 실수치를 인용했으나(14,822 MW), 오늘 브리핑에서는 "파일 확인, 파싱 미완료"로 다시 후퇴. AG2 데이터는 summary JSON에 경로가 명시되어 있고 파일이 존재하므로 파싱 우선순위 유지 필요. 다음 사이클에서 AG2 windcast + load CSV를 Read 대상에 필수 포함.

### (f) Tenaska PTP 8회 연속 미수집 — 정량 MAE 불가 지속
이 항목에 대한 에스컬레이션이 아직 실행되지 않고 있음. 정량 forecast error 추적 없이는 bias calibration이 불가능하므로 pnl-manager/evaluator 경유 확인 요청을 이번 사이클에서 최우선 조치해야 함.
