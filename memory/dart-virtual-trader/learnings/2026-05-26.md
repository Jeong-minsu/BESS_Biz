### 3.2 HE10-11 HIGH Tier at 25 MW — P Overestimation Risk

2026-05-25 self-review identified that first-principles P(DA<RT) estimates for the midday LONG DA block systematically overstate Smartbidder model output by ~7-8pp. Market-analyst cited P=65-66% for HE10-11; Smartbidder model (when returned next day) showed analogous hours at ~58%.

If true P was ~58% (MED zone), these positions were classified as HIGH tier on inflated P. However, given 25 MW sizing (the same as MED tier would produce for this block), the practical consequence to P&L was zero — directional outcome drives the result, not sizing. The classification error did not produce a capital error in this instance.

Lesson: Market-analyst narrative P for LONG DA midday block should be treated as an upper bound, not a point estimate. The Smartbidder file (when available) is the authoritative P source. Do not assign HIGH tier to LONG DA block solely on narrative P > 60% when Smartbidder file is absent and prior calibration shows 7-8pp overestimation bias.
